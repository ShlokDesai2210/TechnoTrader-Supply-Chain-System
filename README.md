# Techno Trader - Electronics Supply Chain Management System

A robust, console-based retail and inventory management system built in Java. Designed to replace manual bookkeeping for electronics retailers, this system bridges the gap between administrative inventory control and the customer purchasing experience.

## 🚀 Key Features

* **Split Restock System:** An advanced inventory feature that allows admins to fulfill total order quantities by splitting purchases across multiple suppliers offering competitive rates.
* **Deep Ledger (Financials):** Automatically records all transaction data (Debits/Credits) and generates a real-time financial report displaying Net Profit or Loss.
* **Dynamic Delivery Allocation:** Evaluates cart contents dynamically to assign appropriate delivery logistics (e.g., dispatching a Van for large items like TVs/PCs, or a Bike for small items).
* **Crash-Proof Navigation:** Utilizes recursive menu rendering and rigorous exception handling (`InputMismatchException`) to ensure a continuous, error-free user session.

## 🛠️ Tech Stack & Concepts
* **Language:** Java (JDK 8+)
* **Core Concepts:** Object-Oriented Programming (Inheritance, Polymorphism), Recursion, Exception Handling, Data Structures (ArrayList).

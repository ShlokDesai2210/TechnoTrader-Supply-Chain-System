# Techno Trader - Electronics Supply Chain Management System

A robust, console-based retail and inventory management system built in Java. [cite_start]Designed to replace manual bookkeeping for electronics retailers, this system bridges the gap between administrative inventory control and the customer purchasing experience[cite: 30, 31, 32].

## 🚀 Key Features

* [cite_start]**Split Restock System:** An advanced inventory feature that allows admins to fulfill total order quantities by splitting purchases across multiple suppliers offering competitive rates[cite: 39, 88].
* [cite_start]**Deep Ledger (Financials):** Automatically records all transaction data (Debits/Credits) and generates a real-time financial report displaying Net Profit or Loss.
* [cite_start]**Dynamic Delivery Allocation:** Evaluates cart contents dynamically to assign appropriate delivery logistics (e.g., dispatching a Van for large items like TVs/PCs, or a Bike for small items)[cite: 42, 95, 96, 97].
* [cite_start]**Crash-Proof Navigation:** Utilizes recursive menu rendering and rigorous exception handling (`InputMismatchException`) to ensure a continuous, error-free user session[cite: 33, 146, 147].

## 🛠️ Tech Stack & Concepts
* [cite_start]**Language:** Java (JDK 8+) [cite: 135]
* [cite_start]**Core Concepts:** Object-Oriented Programming (Inheritance, Polymorphism), Recursion, Exception Handling, Data Structures (ArrayList)[cite: 33, 77, 125, 126].
